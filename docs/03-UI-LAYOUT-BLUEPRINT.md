## Brand Design System

### Colors
```
Primary:     #0468D7  (CTAs, headers, key UI elements)
Accent:      #1A68D3  (Links, secondary buttons, highlights)
Background:  #FFFFFF  (Page backgrounds, cards)
Text Primary: #4A4A4A (Body text, paragraphs)
```

### Typography
```
Heading Font: "Google Sans", sans-serif
Body Font:    "Roboto", sans-serif

H1: 60px / font-bold
H2: 36px / font-semibold
H3: 24px / font-semibold
Body: 20px / font-normal
Small: 14px / font-normal
```

### Spacing
```
Base Unit: 8px

xs: 4px   (0.5 units)
sm: 8px   (1 unit)
md: 16px  (2 units)
lg: 24px  (3 units)
xl: 32px  (4 units)
2xl: 48px (6 units)
3xl: 64px (8 units)
```

### Shape
```
Border Radius: 24px (cards, buttons, inputs)
Border Radius Small: 12px (badges, tags)
Border Radius Full: 9999px (avatars, pills)
```
